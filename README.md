# Selling_Tickets
Информационная система для осуществления учёта продажи билетов на различные мероприятия

В системе реализованы 2 типа пользователей: "Администратор" и "Клиент".
"Клиент" может зарегистрироваться в системе только после подтверждения регистрации "Администратором".

Возможности "Клиента":
- поиск мероприятий по наименованию, типу, дате и времени, месту, цене
- просмотр списка ближайших мероприятий
- вывод и печать календаря мероприятий за выбранный период
- просмотр и печать списка занятых мест
- бронирование свободных мест

Возможности "Администратора":
-  поиск, добавление, изменение и удаление мероприятий
-  поиск, добавление, удаление пользователей
-  изменение статуса пользователя ("Клиент" -> "Администратор")
